# Tг бот "Внутренний справочник контактов"

Бот предназначен для поиска подразделений и сотрудников предприятия.

## Возможности

- поиск сотрудника по ФИО, должности, подразделению;
- поиск подразделения по названию;
- просмотр карточки сотрудника;
- просмотр карточки подразделения и ответственных лиц;
- список подразделений;
- частые и экстренные контакты;
- админ-панель для добавления, редактирования и архивирования записей;
- проверка записей, которые давно не обновлялись

# Настройка

- Создайте базу и тестовые данные:

```
python scripts\seed_db.py
```

- Запустите бота:

```
python bot.py
```

## База данных

Используется SQLite-файл `data/directory.sqlite3`

### Таблица departments

- `id` - идентификатор;
- `name` - название подразделения;
- `description` - краткое описание;
- `internal_phone` - внутренний телефон;
- `phone` - рабочий телефон;
- `email` - почта;
- `location` - кабинет/этаж;
- `work_schedule` - график работы;
- `note` - примечание;
- `is_active` - активна ли запись;
- `updated_at` - дата последнего изменения.

### Таблица employees

- `id` - идентификатор;
- `full_name` - ФИО;
- `position` - должность;
- `department_id` - подразделение;
- `internal_phone` - внутренний телефон;
- `phone` - рабочий телефон;
- `email` - почта;
- `location` - место работы;
- `is_responsible` - ответственное лицо;
- `is_active` - активна ли запись;
- `note` - примечание;
- `updated_at` - дата последнего изменения.
